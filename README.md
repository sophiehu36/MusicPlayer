# MusicPlayer
 
<h2>实现功能</h2>
<ul>
 <li>播放/暂停</li>
 <li>点击按钮切换<b>上一首/下一首</b></li>
 <li>显示专辑图片、歌名、歌手等歌曲信息</li>
 <li>切换歌曲时更新歌曲信息</li>
 <li>显示歌曲时长</li>
 <li>menu按钮切换显示页面</li>
 <li>点击Favourite列表中歌曲名切换歌曲</li>
</ul>
